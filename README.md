# Daelimi (대림이)
Daelim University FAQ Chat bot for Freshman
신입생을 위한 대림대학교 FAQ 챗봇


### 활용 기술 스택
Web Server : Django (Python)  
AI Model : Python (Seq2Seq)  
Android : Java  
Window Application : C# (WinForm)
